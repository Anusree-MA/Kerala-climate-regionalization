## Project Overview
This project applies Principal Component Analysis (PCA) and K-Means clustering to regionalize the climate of Kerala using long-term IMD data (1951–2020).

## Objectives
- Reduce dimensionality of 48 climate variables using PCA
- Identify homogeneous climate zones using K-Means clustering
- Visualize spatial distribution using ArcGIS

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- ArcGIS

## Key Results
- 7 principal components captured 99% variance
- 7 distinct climate clusters identified using Elbow Method
